Порядок чтения файлов:
1_DataPreparation_DatasetsMerge.ipynb:
    Объединение датасетов ga_sessions и ga_hits

2_DataPreparation.ipynb:
    Подготовка данных - анализ, очистка, заполнение пропусков, создание новых фич, скалирование и ohe-трансформирование

3_MachineLearning.ipynb:
    Обучение нескольких моделей, подбор параметров для лучшей из них,
    обучение лучшей модели с выбранными параметрами,
    обучение на сбалансированном датасете

4_pipeline.py:
    Пайплайн обработки данных, обучения лучшей модели и ее сериализация

5_main.py:
    Реализация сервиса


Папка data:
    df_balanced.csv - сбалансированная выборка из финального датасета для обучения
    Остальные датасеты не влезли по объему в разрешенное пространство, 
	но их описание есть в ноутбуках.

Папка model:
    cars_pipe.pkl - сериализованный пайплайн обработки данных с моделью
    freq.pkl - сериализованный файл с частотами полей utm_campain и utm_source

Папка test:
    4 json-файла для проверки api (значение target и session_id прописаны в названии)
