Порядок чтения файлов:
1_DatasetFromParquet.ipynb:
    объединение датасета из файлов parquet

2_DatasetAggregation.ipynb:
    Ohe-трансформирование и последовательная аггрегация данных по фрагментам

3_FeatureEngineering.ipynb:
    Генерация новых фичей

4_ModelSelection_and_Training.ipynb:
    Обучение нескольких моделей, подбор параметров для них,
    обучение лучших моделей с выбранными параметрами,
    усреднение результатов

4_pipeline.py:
    Пайплайн обработки данных, обучение моделей и сериализация

5_pipeline_pickle_on_test.py:
    Проверка работы сериализованного пайплайна с моделью на тестовой выборке
   

Папка model:
    bank_pipe.zip  - архив с файлом bank_pipe.pkl:
	сериализованный пайплайн обработки данных с моделью
